- test/api-test.sh — интерактивный тестовый скрипт (bash + curl), прогоняет 30+ проверок полного сценария:

Шаг	Что проверяет
- 1	Логин админа
- 2-3	Админ создаёт кастомный класс, расу, стат, предмет
- 4-5	Регистрация игрока и ГМ
- 6-9	Игрок: создание персонажа, просмотр статов/инвентаря, экипировка, апдейт
- 10-11	ГМ: создание команды, игрок вступает по invite-коду
- 12-13	ГМ: просмотр персонажей команды, редактирование стата
- 14	Негативные тесты (ГМ не может менять имя/инвентарь, админ не может создать персонажа, игрок не может в /admin)
- 15-16	Регенерация invite-кода, админ просматривает всех
- 17	Удаление персонажа

### Запуск:

- docker-compose up -d
- подождать ~15 секунд
- bash test/api-test.sh
